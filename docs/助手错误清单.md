# 助手错误清单

## ❌ 常犯错误

### 1. 目录错误
- **错误**: 在 `backend/` 目录执行命令
- **解决**: 始终在项目根目录 `D:\Projects\NewsMind` 工作

### 2. 文件位置错误  
- **错误**: 在根目录创建测试文件
- **解决**: 测试脚本→`test_results/`，启动脚本→`backend/`

### 3. 路径错误
- **错误**: `python test_server.py`
- **解决**: `python backend/test_server.py`

### 4. 重复文件
- **错误**: 创建多个相似启动脚本
- **解决**: 统一使用 `backend/start_server.py`

### 5. 环境混乱
- **错误**: 虚拟环境和全局环境混用
- **解决**: 明确使用项目虚拟环境

## ✅ 正确做法

# 工作目录
PS D:\Projects\NewsMind>

# 启动后端
